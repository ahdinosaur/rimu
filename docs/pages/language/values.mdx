# Rimu values

## Value

### Null

```rimu
null
```

### Boolean

```rimu
true
```

```rimu
false
```

### Number

All numbers are ["Big" Decimals](https://crates.io/crates/rust_decimal), not floats or integers.

```rimu
10
```

```rimu
10.99
```

### String

```rimu
"hello world"
```

```rimu
"they said \"hello world\""
```

### List

```rimu
["hello", null, "world"]
```

### Object

> Unquoted keys must conform to [Identifier](#Identifier) regex: `^[a-zA-Z_][a-zA-Z0-9_]*$`
>
> Quoted keys can be any valid Unicode.


```rimu
{
  a: "apple",
  b: "bear",
}
```

```rimu
{
  "a": "apple",
  "b": "bear",
}
```

### Function

```rimu
(a, b) => a + b
```
