# Start: Command-line

## `rimu-cli`

To use Rimu to render data templates:

```shell
cargo install rimu-cli
```

```shell
rimu-cli
```

```txt
Usage: rimu-cli [OPTIONS] --input INPUT

Options:
  -i, --input INPUT
  -e, --env ENV
  -o, --output OUTPUT  [default: -]
  -f, --format FORMAT  [default: yaml]
  -h, --help             Print help
  -V, --version          Print version
```

## `rimu-repl`

To use Rimu as an expression REPL (Read-Eval-Print-Loop):

```shell
cargo install rimu-repl
```

```shell
rimu-repl
```
